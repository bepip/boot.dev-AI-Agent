# AI-Agent

## Overview

This repository contains my work and progress from the "Build an AI Agent in Python" guided project offered by [boot.dev](https://www.boot.dev/courses/build-ai-agent-python). The course focuses on building a tiny coding agent that can read files, edit code, run Python, and loop on feedback. 

**Note:** This repository is a result of following the guided project.

## Goal of project
* Understanding how AI tools actually work under the hood.
